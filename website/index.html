Why choose Vector?

Freedom of Open Source

Vector is Apache licensed open source code and benefits from the vibrant contributing community of Matrix.org. This means everyone has the flexibility and control to audit and extend the code with new features.

What's yours is yours

Vector provides state of the art end to end encryption over a decentralised architecture thanks to the Matrix.org open standard. Only you have the key to your data. Run your own Matrix server for total control.

Everything together

From Github to Jira, from IRC to Slack: Vector connects to an ever increasing range of applications and networks through the Matrix.org open standard. Easily connect to your all contacts: they don't even need to create an account!

Find your focus

We all know how distracting messaging apps can be. Vector provides an unobtrusive interface with simple yet powerful control over your notifications, making it easy to tune in or tune out on your own terms.
